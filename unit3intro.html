Unit 3: Numerical Methods

This section explores methods that help find solutions when exact answers are difficult or impossible to calculate.

Key Concepts:

Interpolation:

  • Lagrange Interpolation: Estimates unknown values between known data points by creating a polynomial that fits those points.
  • Inverse Interpolation: Finds the value of an independent variable when the dependent variable is given.

Finite Differences:

  • Forward Differences: Calculates the difference between successive data points.
  • Backward Differences: Similar to forward differences but works in the opposite direction.
  • Divided Differences: Handles unevenly spaced data points by calculating differences between them.

Difference Tables:

  • Forward and Backward Difference Tables: Organize and simplify the process of calculating differences for interpolation.
  • Divided Difference Table: Organizes divided differences for interpolation when data points are not equally spaced.

Newton's Methods of Interpolation:

  • Newton's Forward Difference Method: Estimates values when data points are evenly spaced.
  • Newton's Backward Difference Method: Used when values at the end of the data set are more important.
  • Newton's Divided Difference Method: Used for unevenly spaced data points.

Numerical Integration:

  • Newton-Cotes Integration Formula: Approximates the area under a curve for integration.
  • Trapezoidal Rule: Estimates the area under a curve by approximating it as a series of trapezoids.
